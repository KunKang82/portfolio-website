Case Study for myFlix full-stack project

Overview

myFlix is a web application built using the MERN stack. It offers users comprehensive access to details about movies, directors, and genres. Through myFlix, users can register an account, manage their personal information, and curate a list of their favorite movies for easy access.

Purpose & Context

myFlix served as a personal project developed during my web development course at CareerFoundry. Its primary aim was to showcase my proficiency in full-stack JavaScript development.

Objective

The objective of the project was to create an ambitious full-stack project to enhance my professional portfolio. The primary goal was to develop both the server-side and client-side components of the application entirely from scratch.

Click to enlarge

Approach

Server-Side

I developed a RESTful API using Node.js and Express, which communicates with a non-relational database (MongoDB). The API supports standard HTTP methods such as GET and POST. CRUD methods are employed to retrieve and store data in the database. Additionally, the API delivers movie information in JSON format.

To test the API, I utilized Postman. Furthermore, I incorporated user authentication and authorization mechanisms, including basic HTTP authentication and JWT authentication.

Client-Side

After finalizing the API, I proceeded to construct the client-side interface for users to interact with the server-side. The interface is a single-page, responsive application built with React and React-Redux. It offers various interface views including the main view displaying a list of all movies, single movie view providing details about a single movie and enabling users to add it to their favorites, login view, registration view, and profile view allowing users to update their data and favorites list.

Challenges

This project presented both my favorite and most challenging experiences. I enjoyed building the API and mastering database structures, as well as becoming proficient with the terminal. However, developing the client-side posed challenges, requiring time to grasp React and Redux concepts and achieve desired outcomes. With guidance from my tutor and a pair-programming session with an experienced developer, I successfully overcame these obstacles.

Duration

This project spanned approximately 2 months, with 1 month dedicated to the backend and another month focused on the frontend development.
